# Фотографии Манибандха Прабху

Положите сюда 4 фотографии с именами:

- `1.jpg` — главный портрет (используется в hero на главной и на странице входа)
- `2.jpg`, `3.jpg`, `4.jpg` — галерея на главной

Фотографии отображаются в чёрно-белом виде автоматически (CSS-фильтр `photo-bw`),
исходные файлы можно класть цветными. При желании можно сохранить их уже в ч/б.
